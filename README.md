# The Gizra Way

> The Gizra Way is a set of best practice and standardization for development team and project managers that aim to deliver 100% successful project.

## Installation

```bash
npm install
bower install
bundle install
```

* Serve locally: `gulp`
* Build and deploy: `gulp publish && gulp deploy`
